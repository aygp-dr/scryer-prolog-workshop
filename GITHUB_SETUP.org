* Setup Methods

** Method 1: Using GitHub CLI (Automated)

#+BEGIN_SRC bash
# Install gh CLI if needed
sudo pkg install gh  # FreeBSD
# brew install gh    # macOS

# Authenticate
gh auth login

# Run the setup script
./github-setup.sh
#+END_SRC

** Method 2: Manual Setup via Web Interface

1. Go to https://github.com/new or https://github.com/organizations/aygp-dr/repositories/new

2. *Configure repository:*
   - Repository name: =scryer-prolog-workshop=
   - Description: =A comprehensive workshop for learning Scryer Prolog with literate programming using Org-mode. Includes automated setup, examples, and LSP support for Emacs.=
   - Visibility: *Public*
   - ❌ Do NOT initialize with README, .gitignore, or license (we already have these)

3. *Click "Create repository"*

4. *Add topics:*
   - Click the ⚙️ gear icon next to "About"
   - In the "Topics" field, add: =prolog=, =scryer-prolog=, =literate-programming=, =org-mode=, =rust=
   - Click "Save changes"

5. *Push your local repository:*
   #+BEGIN_SRC bash
   git remote add origin git@github.com:aygp-dr/scryer-prolog-workshop.git
   git push -u origin main
   #+END_SRC

** Method 3: Using GitHub CLI Manually

#+BEGIN_SRC bash
# Create repository
gh repo create aygp-dr/scryer-prolog-workshop \
  --public \
  --description "A comprehensive workshop for learning Scryer Prolog with literate programming using Org-mode. Includes automated setup, examples, and LSP support for Emacs." \
  --source=. \
  --remote=origin \
  --push

# Add topics
gh api repos/aygp-dr/scryer-prolog-workshop/topics \
  --method PUT \
  --field names[]="prolog" \
  --field names[]="scryer-prolog" \
  --field names[]="literate-programming" \
  --field names[]="org-mode" \
  --field names[]="rust"
#+END_SRC
